# Prompt Hub

import PromptFiles from 'components/PromptFiles';

Il Prompt Hub è una raccolta di prompt utili per testare le capacità dei LLM su una varietà di competenze fondamentali e compiti complessi. Ci auguriamo che il Prompt Hub ti aiuti a scoprire modi interessanti per sfruttare, sperimentare e creare con i modelli linguistici. Incoraggiamo e accogliamo con piacere i contributi della comunità di ricerca sull’IA e degli sviluppatori.
